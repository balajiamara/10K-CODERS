What is Internet

The Internet is a global network of computers and devices that are interconnected, allowing them to communicate and share information. It uses a standardized set of rules called Internet Protocol (IP) and Transmission Control Protocol (TCP) to transmit data.

Example: Browsing websites,Online Shopping

What is Browser

A browser (or web browser) is a software application used to access and view websites on the Internet.

Example: Google Chrome, Mozilla Firefox, Microsoft Edge, Safari

What is URL

A URL (Uniform Resource Locator) is the address you type into a browser to visit a specific webpage or online resource.

Example: https://www.google.com

What is Domain

A domain (or domain name) is the human-readable address of a website on the Internet.

Sub Domain: www
Second-Level Domain (SLD): google, wikipedia, etc.
Top-Level Domain (TLD):.com, .org, .in, etc.

What is http

HTTP stands for HyperText Transfer Protocol. It is the set of rules used by web browsers and web servers to communicate and transfer data over the Internet

Example: http://www.example.com

What is https

HTTPS stands for HyperText Transfer Protocol Secure. It is the secure version of HTTP, the protocol used to transfer data between your browser and websites

Example: https://www.example.com

Difference between http and https

We Use HTTP for basic, public content.
We Use HTTPS when security and privacy are important.

Example:http://www.example.com → Not secure
https://www.example.com → Secure

What is IP Address

An IP address (Internet Protocol address) is a unique number assigned to every device connected to the Internet. It acts like a home address for your device

Example: 192.168.1.1

Difference between Programming Language and Mark-up Language

A programming language and a markup language serve distinct roles in computing. A programming language is used to write instructions that a computer can execute to perform specific tasks, such as creating applications, manipulating data, or controlling hardware.
